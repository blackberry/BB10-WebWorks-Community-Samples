Toasts
Some Notes
Toasts really have nothing to do with BBM, but they're freaking awsesome, so I threw one in here.
The code I actually used is a lot more complicated than toast code needs to be, so I included a basic toasts here as well.

The Code:
function showCustomToast() {
var message = "Butter me?",
buttonText = "Yes!",
toastId,
onButtonSelected = function () {
blackberry.ui.toast.show('Buttered!')
},
onToastDismissed = function () {},
options = {
buttonText : buttonText,
dismissCallback : onToastDismissed,
buttonCallback : onButtonSelected
};

toastId = blackberry.ui.toast.show(message, options);
}

Simple Sample:
blackberry.ui.toast.show('Hello World')
