Start A Chat
Some Notes
As of now, the only difference between starting a chat and starting a chat with a message ready to send is the data parameter. That might change in the future.
This invoke allows the user of your app to select the contact they want to chat with.

Start A Chat:
function startChat() { blackberry.invoke.invoke({
target: 'sys.bbm.sharehandler',
action: 'bb.action.SHARE',
data: '',
mimeType: 'text/plain'
})
}

Chat with Text:
function startChatWText() {
blackberry.invoke.invoke({
target: 'sys.bbm.sharehandler',
action: 'bb.action.SHARE',
data: 'Have you checked out the KISS BBM Integration app?',
mimeType: 'text/plain'
})
}

Do it!
onclick="startChat()"
onclick="startChatWText()"
