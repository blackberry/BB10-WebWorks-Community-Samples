Add A PIN
Some Notes
This function allows a new BBM contact to be added. You can manually insert the pin, or you can have it retrieved from an element in the document like the Personal Message example. However, the code used to Chat with a Designate PIN is probably better suited to most purposes, as it adds the contact first if they don't exist in the BBM Contact List.

The Code:
function addMe() {
blackberry.invoke.invoke({
action: 'bb.action.INVITEBBM',
uri: 'pin:32693EF6'
})
}

Do it!
onclick="addMe()"
