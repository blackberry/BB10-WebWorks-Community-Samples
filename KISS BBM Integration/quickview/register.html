Registration
Some Notes
Registration is a prerequisite for several other BBM integrated features.
This code is included twice in this sample app. The register() function is tied to a button, and the onLoadFunctions version runs when you load a page in a bbUI app.
If you use the onLoadFunctions version, make sure that you insert onLoadFunctions(); into the ondomready section of the index.html.

The Code:
function register() {
blackberry.event.addEventListener('onaccesschanged', function (accessible, status) {
if (status === 'unregistered') {
blackberry.bbm.platform.register({
uuid: '33490f91-ad95-4ba9-82c4-33f6ad69fbbc'
});
} else if (status === 'allowed') {
bbm.registered = accessible;
}
}, false)
}

Do it!
onclick="register()"
