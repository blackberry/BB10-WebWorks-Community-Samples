Update Status
Some Notes
The user's BBM Status can be updated to either Available or Busy, and an optional status message can be posted.

Available Status:
function setStatusA() {
blackberry.bbm.platform.self.setStatus('available', 'See? I\'m available!')
}

Busy Status:
function setStatusB() {
blackberry.bbm.platform.self.setStatus('busy', 'And now I\'m busy.')
}

Do it!
onclick="setStatusA()"
onclick="setStatusB()"
