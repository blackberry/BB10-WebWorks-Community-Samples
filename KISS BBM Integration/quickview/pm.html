Personal Message
Some Notes
Instructions for setting a Personal Message and for setting up a user customizable personal message are shown here.
If you use a customizable one, make sure you actually have the element needed to call the data from (obviously).

Regular Personal Message:
function setPM() {
blackberry.bbm.platform.self.setPersonalMessage('I set my status using the KISS BBM Integration app.')
}

Custom Personal Message:
function setPMW() {
blackberry.bbm.platform.self.setPersonalMessage('My status is now ' + document.getElementById('custommessage').value)
}

Do it!
onclick="setPM()"
onclick="setPMW()"
