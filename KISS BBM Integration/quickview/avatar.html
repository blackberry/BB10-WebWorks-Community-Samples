Set The Avatar
Some Notes
This lets you set an avatar from an image packaged with the app.
You could also invoke the file manager and allow a picture to be chosen from there, but I honestly couldn't think of a use for that.

The Code:
function setAvatar() {
blackberry.invoke.invoke({
target: 'sys.bbm.imagehandler',
action: 'bb.action.SET',
uri: 'local:///images/avatar.png'
});
}

Do it!
onclick="setAvatar()"
