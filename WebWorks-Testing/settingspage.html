The WebWorks Testing App
The WebWorks Testing app was originally a simple app that launched html files from your BlackBerry's internal memory. At this point, it has expanded a fair bit beyond it's humble origins in features, but still (in my opinion) fulfills its original goal of making WebWorks apps even easier to build and test.

Setup:
First, decide where you'd like to load your WebWorks code from. If you'd like to load it from an external URL (like a home server, a personal webpage, etc), just type the domain name or IP address into the box with the "http://" in front of it, and click the "Set Web Location" button. If you'd like to load it from the BlackBerry filesystem, just type where you're going to store the files into the box with "file:///accounts/1000/shared/" in front of it. For example, if you wanted to use the same directory as the original WebWorks Testing app, you could just type misc/Testing/index.html into that box.
Next, you choose how you'd like to return to this settings page (so you can change the location of your testing files or clear the WebWorks Testing app's localStorage cache). You do this by selecting one of the checkboxes. Landscape will open this settings screen if the BlackBerry is not in Landscape mode, portrait will open it if your BlackBerry is not in portrait mode. If you select Either and leave these settings, you'll never see them again unless you delete/reinstall the app. And if you select Neither, you'll be returned to this screen next time you launch the app.
If you have any questions, feel free to email me or post on my Github account (github.com/SCrid2000).
