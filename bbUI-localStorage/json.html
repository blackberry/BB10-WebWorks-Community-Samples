JSON
Sometimes you might want to store more than one item in localStorage; there's no real reason to have several hundred localStorage values. You can use JSON to store and retrieve multiple items from localStorage.
Stringify
Saving a JSON string is quite easy. Here's the sample code for the button below:

localStorage.setItem('jsonString', JSON.stringify({
option1: 'Hello World',
option2: 'Goodbye World',
option3: document.getElementById('JSONbox').value,
}))

The id of the box below is JSONbox; whatever you enter as the content of it will be saved.
Stringify!
Parse the String
Once you've stored a string, it wouldn't be much good if you couldn't retrieve a specific item. Retrieval is quite simple as well.
To retrieve a specific saved value, just execute the following:
JSON.parse(localStorage.getItem('yourvalue')).theStringName

For this example, you could run any of the following:

JSON.parse(localStorage.getItem('jsonString')).option1
JSON.parse(localStorage.getItem('jsonString')).option2
JSON.parse(localStorage.getItem('jsonString')).option3
Although in this example I used BlackBerry Toasts, you could use the JSON string to populate pretty much anything, like a text box:

option1
option2
option3
Anytime you need to store multiple values, localStorage + JSON is the way to go.
