You don't have a localStorage value set.
You see this because you don't have a localStorage value set (it returned "null"). When you clicked the Redirect option, it ran a javascript function found in the custom.js file called "redirect." Redirect is quite similar to the Welcome javascript, and looks like this:
function redirect() {
if (localStorage.getItem('redirect') === null) {
bb.pushScreen('redirect/one.html', 'one'); }
else if (localStorage.getItem('redirect') === 'something') {
bb.pushScreen('redirect/three.html', 'three'); }
else
bb.pushScreen('redirect/two.html', 'two');
}

Click one of the buttons below to set the localStorage value to a value that isn't set in the custom.js javascript file. Because none of the below values are attached to a condition, it really doesn't matter which you select (it's an example).

Next time you launch Redirect, a different page will be loaded.
Set localStorage to "helloworld" and return to the main screen.
Set localStorage to "goodbyeworld" and return to the main screen.
Set localStorage to "okIgetit" and return to the main screen.

If you're asking yourself, why is this useful? the answer is that you can use this to have the same button do different functions, based on user preferences.
