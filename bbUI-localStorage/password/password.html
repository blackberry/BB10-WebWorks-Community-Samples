If you tried entering a password before you set one, you might have noticed that you get redirected to the Set Password page; this is by design, and is because the function passprotect has the line:
if (localStorage.getItem('password') == null) {bb.pushScreen('password/setpassword.html','setpass')
If you would prefer to have nothing happen if a password is not set, you can change that line to:
if (localStorage.getItem('password') == null) {}
While this feature works for a sandboxed app, do NOT use this for a regular webpage or anything where localStorage can be viewed! This method stored the password in plain text, and if used outside a secured environemnt such as a PlayBook WebWorks app, your password WILL NOT be safe!
