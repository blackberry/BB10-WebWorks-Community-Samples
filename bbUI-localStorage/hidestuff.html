Hiding Things
Style="display:none"
You probably know that you can hide things with css display: none. You can have this done automatically with localStorage.

Example:
if (localStorage.getItem('hideinfo') != null) {document.getElementById('hideinfo').style.display = 'none'}
This script will hide the Background Info section on the main page of this app. Press the button below to enable it.
Hide the Background Info.

If you'd live to restore it, just press this:
localStorage.removeItem('hideinfo')
